# Home Assistant Add-Ons

## Hassio Access Point

Forked to remove DSSS_CCK-40 from hostapd.conf. This setting may cause driver issues with:

* mt76x2u

Use your hass.io host as a WiFi access point - perfect for off-grid and security focused installations.

## Installation

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fcabinlab%2Fhassio-addons)

Or manually add
`https://github.com/cabinlab/hassio-addons` to your Home Assistant addon repositories list. See [instructions](https://www.home-assistant.io/hassio/installing_third_party_addons/) on the Home Assistant website.
